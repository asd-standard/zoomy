Contribution guidelines
=======================

Every new class or method must have it's docstring with 

Constructor :
	class.method('parameter 1','parameter 2',...)
Parameters :
	parameter 1['type1'], parameter 2['type2'], ...

class.method('parameter 1', 'parameter 2') --> Returned_Obj 1['type1'], ecc 

everything it's done within the method has to be explained:

Internal and global variable manipulation ecc
